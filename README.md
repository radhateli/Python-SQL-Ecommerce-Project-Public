# 🛒 E-Commerce Data Analysis using SQL & Python

## 📌 Project Overview

This project analyzes an E-Commerce dataset using **MySQL, SQL, and Python** to extract meaningful business insights related to customers, orders, sales, and payments.

The dataset was imported into MySQL and queried using SQL, while Python was used for database connectivity and result extraction.

---

## 🎯 Business Questions Solved

### 1️⃣ List all unique cities where customers are located

**Objective:** Identify the geographical distribution of customers.

**SQL Concepts Used:**
- DISTINCT
- SELECT

---

### 2️⃣ Count the number of orders placed in 2017

**Objective:** Measure yearly order volume.

**Result:**
- 📦 Total Orders in 2017: **45,101**

**SQL Concepts Used:**
- COUNT()
- Date Filtering
- STR_TO_DATE()

---

### 3️⃣ Find the total sales per category

**Objective:** Identify top-performing product categories by revenue.

**SQL Concepts Used:**
- SUM()
- GROUP BY
- JOIN Operations
- Aggregations

---

### 4️⃣ Calculate the percentage of orders paid in installments

**Objective:** Understand customer payment behavior.

**SQL Concepts Used:**
- COUNT()
- CASE Statements
- Percentage Calculations

---

### 5️⃣ Count the number of customers from each state

**Objective:** Analyze customer distribution across states.

**SQL Concepts Used:**
- GROUP BY
- COUNT()
- Aggregation Functions
